# Security Policy

## Supported Versions

Currently, the following versions of `@express-route-cache` are being supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| v0.1.x  | :white_check_mark: |

## Reporting a Vulnerability

Please do not open a public issue if you've discovered a security vulnerability.
Instead, please contact the maintainer privately outlining the exact vector, steps to reproduce, and impact.

- **LinkedIn**: [Yatharth Lakhate](https://www.linkedin.com/in/yatharth-lakhate/)
- **X (Twitter)**: [@Yatharth_L](https://x.com/Yatharth_L)

We will respond promptly to acknowledge the report and will work quickly with you to issue a patch to NPM and GitHub. We'll ensure the vulnerability is addressed securely before any public disclosure occurs.
